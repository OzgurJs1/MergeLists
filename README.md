# MergeLists
Bu kod, iki bağlı listenin elemanlarını birleştirir ve yeni bir bağlı liste oluşturur. İlk olarak, iki bağlı listenin başlangıç düğümleri (head) fonksiyona parametre olarak geçilir. Sonra, ilk bağlı listenin son düğümü, ikinci bağlı listenin başlangıç düğümüne bağlanır ve böylece iki liste tek bir bağlı liste haline getirilir. Son olarak, birleştirilmiş bağlı liste ekrana yazdırılır.
